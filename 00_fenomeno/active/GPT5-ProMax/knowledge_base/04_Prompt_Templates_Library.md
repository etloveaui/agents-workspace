# 프롬프트 템플릿 라이브러리

## 🎯 템플릿 사용 가이드

이 라이브러리는 GPT5의 특성에 최적화된 재사용 가능한 프롬프트 템플릿을 제공합니다. 각 템플릿은 `reasoning_effort`, `verbosity`, auto-routing 최적화가 내장되어 있습니다.

### 템플릿 선택 기준
- **Basic**: 일반적 작업 (reasoning_effort: minimal-low)
- **Professional**: 업무용 고품질 (reasoning_effort: medium)  
- **Expert**: 전문가 수준 (reasoning_effort: high)
- **Research**: 심층 분석 (reasoning_effort: high, thinking 모드 유도)

## 🚀 1. 기본 템플릿 (Basic Templates)

### A. 일반 정보 요청
```markdown
# 정보 요청 템플릿

## 📋 요청 내용
[구체적으로 알고 싶은 정보]

## 🎯 목적  
[정보가 필요한 이유나 활용 방안]

## 📊 원하는 형태
- [ ] 간단한 요약
- [ ] 상세한 설명  
- [ ] 단계별 가이드
- [ ] 비교 분석
- [ ] 기타: [구체화]

## 📏 상세도
- [ ] 핵심만 간단히 (verbosity: low)
- [ ] 적절한 설명과 함께 (verbosity: medium)
- [ ] 매우 상세하게 (verbosity: high)

[GPT5 Auto-routing: 자동으로 최적 모델 선택됩니다]
```

## 💼 2. 전문 업무 템플릿 (Professional Templates)

### A. 비즈니스 분석 템플릿
```markdown
# 비즈니스 분석 프롬프트

## 🏢 분석 대상
[회사명, 제품, 서비스, 시장 등]

## 📊 분석 프레임워크
다음 관점에서 체계적으로 분석해주세요:

### 1. 현황 분석 (Current State)
- 시장 포지션과 경쟁 우위
- 주요 성과 지표와 트렌드
- 강점과 약점 식별

### 2. 환경 분석 (Environmental Scan)
- 시장 동향과 기회 요인
- 위협 요소와 리스크
- 규제 환경과 정책 변화

### 3. 전략적 함의 (Strategic Implications)
- 핵심 도전과제 식별
- 성장 기회와 우선순위
- 권장 액션 플랜

## 🎯 출력 요구사항
- **Executive Summary**: 핵심 발견사항 3-5줄
- **상세 분석**: 각 섹션별 심층 분석
- **시각화**: 가능한 경우 표나 차트 형태 제안
- **실행 계획**: 구체적이고 측정 가능한 다음 단계

## ⚡ GPT5 최적화
"Think strategically about the complex business implications and provide expert-level analysis"
```

```markdown
- **성공 기준**: [측정 가능한 완료 조건]

## 🎯 계획 수립 범위

2. **일정 계획**: 마일스톤과 주요 일정
3. **리소스 계획**: 인력, 도구, 예산 배분
4. **리스크 관리**: 주요 위험요소와 대응방안
5. **커뮤니케이션 계획**: 보고 체계와 의사결정 프로세스
6. **품질 관리**: 품질 기준과 검증 방법

## ⚡ GPT5 최적화  
"Develop a comprehensive project plan considering all critical success factors and potential risks"
```

```markdown
- 대안 기술과 트레이드오프
- 라이브러리/프레임워크 선택

- 모니터링 및 로깅

- 규정 준수 고려사항

"Design a robust, scalable, and secure system architecture considering all technical and business constraints"
```

### B. 투자 분석 템플릿
```markdown
# 투자 기회 분석 템플릿

## 💰 투자 개요
- **투자 대상**: [기업명, 프로젝트명, 자산 유형]
- **투자 규모**: [예상 투자 금액]
- **투자 기간**: [예상 보유 기간]
- **투자 목적**: [수익형/성장형/안정형]

## 📊 분석 프레임워크
다음 관점에서 전문적인 투자 분석을 수행해주세요:

### 1. 재무 분석 (Financial Analysis)
- **수익성 지표**: ROI, ROE, ROIC, 매출 성장률
- **안정성 지표**: 부채비율, 유동비율, 이자보상배율  
- **효율성 지표**: 자산회전율, 재고회전율
- **시장 지표**: PER, PBR, EV/EBITDA

### 2. 사업 분석 (Business Analysis)  
- **비즈니스 모델**: 수익 구조와 핵심 동력
- **경쟁 우위**: 해자(moat)와 차별화 요소
- **시장 포지션**: 시장 점유율과 성장 가능성
- **경영진 평가**: 리더십과 전략 실행 능력

### 3. 리스크 분석 (Risk Assessment)
- **사업 리스크**: 산업, 경쟁, 규제 리스크
- **재무 리스크**: 유동성, 신용, 환율 리스크  
- **운영 리스크**: 키맨, 기술, 공급망 리스크
- **시장 리스크**: 금리, 인플레이션, 경기순환

### 4. 밸류에이션 (Valuation)
- **절대가치 평가**: DCF, DDM 모델링
- **상대가치 평가**: 동종업계 멀티플 비교
- **시나리오 분석**: 낙관/기준/비관 시나리오
- **민감도 분석**: 주요 변수 변동 영향

## 🎯 투자 의사결정 지원
- **투자 권고**: Buy/Hold/Sell 추천과 근거
- **목표 수익률**: 예상 수익률과 달성 확률
- **포트폴리오 적합성**: 기존 포트폴리오와의 조화
- **모니터링 지표**: 추적해야 할 핵심 지표들

## ⚡ GPT5 최적화
"Provide expert-level investment analysis with rigorous financial modeling and risk assessment"
```

## 🔍 4. 연구 분석 템플릿 (Research Templates)

### A. 경쟁사 분석
```markdown
# 경쟁사 분석 템플릿

## 🏢 분석 대상
- **주요 경쟁사**: [3-5개 직접 경쟁사]
- **간접 경쟁사**: [대체재, 신규 진입자]
- **벤치마크 대상**: [업계 선도 기업]

## 📊 분석 프레임워크
각 경쟁사에 대해 다음을 체계적으로 분석해주세요:

### 1. 기업 개요
- 연혁과 주요 마일스톤
- 비즈니스 모델과 수익 구조
- 조직 구조와 핵심 인재

### 2. 제품/서비스 포트폴리오
- 주력 제품과 서비스 라인업
- 혁신 역량과 R&D 투자
- 품질과 고객 만족도

### 3. 시장 전략
- 타겟 시장과 고객 세그먼트
- 마케팅 믹스 전략
- 유통 채널과 파트너십

### 4. 재무 성과
- 매출 및 수익성 트렌드  
- 시장 점유율 변화
- 투자 활동과 재무 건전성

### 5. 강점과 약점
- 핵심 경쟁 우위
- 취약점과 개선 기회
- 위협 요소

## 🎯 통합 분석
- **경쟁 지형도**: 포지셔닝 맵과 세력 분포
- **성공 요인**: 업계 내 성공하는 기업의 공통점
- **기회와 위협**: 우리 기업 관점에서의 함의
- **전략적 권고**: 경쟁 우위 확보 방안

## ⚡ GPT5 최적화
"Conduct thorough competitive intelligence analysis to identify strategic opportunities and threats"
```

### B. 시장 조사 연구
```markdown
# 시장 조사 연구 템플릿

## 🎯 연구 목적
- **연구 질문**: [핵심적으로 답하고자 하는 질문]
- **의사결정 지원**: [이 연구로 내릴 결정]
- **이해관계자**: [연구 결과 활용 주체]

## 🔍 시장 정의
- **제품/서비스 범위**: [정확한 시장 경계]
- **지리적 범위**: [타겟 지역]
- **시간적 범위**: [분석 기간]
- **고객 범위**: [타겟 고객군]

## 📊 시장 분석 요청
다음 영역에 대한 종합적인 시장 조사를 수행해주세요:

### 1. 시장 규모 및 성장성
- **TAM (Total Addressable Market)**: 전체 시장 규모
- **SAM (Serviceable Addressable Market)**: 실제 접근 가능 시장
- **SOM (Serviceable Obtainable Market)**: 현실적 확보 가능 시장
- **성장률**: 과거 트렌드와 미래 전망
- **성장 동력**: 시장 성장을 견인하는 요인들

### 2. 고객 분석
- **고객 세그멘테이션**: 주요 고객 그룹 분류
- **구매 행동**: 의사결정 과정과 구매 패턴
- **니즈와 Pain Points**: 충족되지 않는 욕구
- **가격 민감도**: 지불 의향과 가격 탄력성
- **채널 선호도**: 선호하는 구매/접촉 채널

### 3. 경쟁 환경
- **시장 구조**: 집중도와 경쟁 강도
- **주요 플레이어**: 시장 지배력과 전략
- **진입 장벽**: 신규 진입의 어려움
- **대체재 위협**: 기존 솔루션을 대체할 가능성
- **협상력**: 공급업체/구매자 영향력

### 4. 트렌드 및 기회
- **기술 트렌드**: 혁신 기술의 영향
- **규제 변화**: 정책/규제 환경 변화
```
